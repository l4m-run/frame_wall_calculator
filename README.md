# Калькулятор каркасной стены

Веб-приложение для расчета материалов каркасной стены с визуализацией конструкции в SVG.

Задаете параметры стены, добавляете проемы (двери/окна) - получаете чертеж, спецификацию, инструкцию по сборке и смету.

## Возможности

- Расчет количества стоек, обвязки, перемычек, ригелей, подоконников
- SVG-чертеж стены с размерными линиями и кумулятивной шкалой
- Оптимальный раскрой досок для проемов (визуализация)
- Расчет гвоздей с разбивкой по узлам
- Смета при указании стоимости доски
- Инструкция по сборке с рекомендациями
- Адаптивный интерфейс

## Стек

Чистый HTML + CSS + JavaScript, без зависимостей. Шрифт Inter (Google Fonts).

## Запуск

Открыть `index.html` в браузере. Сервер не требуется.

## Входные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Длина стены | 6000 мм | Общая длина стены |
| Высота стены | 2800 мм | Общая высота (включая обвязки) |
| Шаг стоек | 575 мм | Расстояние между осями (центрами) стоек |
| Ширина доски | 150 мм | Глубина сечения доски |
| Толщина доски | 50 мм | Толщина сечения доски |
| Двойная верхняя обвязка | Да | Второй слой верхней обвязки со смещением стыков |
| Ригель | Нет | Доска на ребро, врезается в выборки стоек под верхней обвязкой |
| Стоимость доски 6м | - | Опционально, для формирования сметы |

### Проемы

Для каждого проема задается:
- **Тип** - дверь или окно (у дверей отступ снизу всегда 0)
- **Ширина, высота** - размеры проема в мм
- **Отступ слева** - от левого края стены до левого края проема
- **Отступ снизу** - от верха нижней обвязки до низа проема
- **Ригель** - опциональная доска на ребро над перемычкой проема

## Алгоритм расчета

### Высота стоек

```
высота_стоек = высота_стены - толщина_доски (нижн. обвязка) - толщина_доски × N (верхн. обвязка)
```

где N = 1 или 2 (одинарная/двойная верхняя обвязка).

### Расстановка стоек

1. Первая стойка ставится на позицию 0 (левый край)
2. Каждая следующая через `шаг_стоек` от предыдущей
3. Последняя стойка всегда на позиции `длина_стены - толщина_доски` (правый край)
4. Если предпоследняя стойка совпадает с крайней - дубль не добавляется

Шаг задается **по осям** стоек. Чистый просвет между стойками = шаг - толщина доски.

### Проемы

Каждый проем добавляет:
- **2 стойки обрамления** - усиление по бокам проема
- **1 перемычку** - доска плашмя над проемом (всегда)
- **1 ригель проема** - доска на ребро над перемычкой (опционально)
- **1 подоконник** - доска плашмя под проемом (только для окон)

### Расчет 6-метровых досок

**Стойки**: из одной 6м доски нарезается `floor(6000 / высота_стоек)` стоек. Общее количество досок = `ceil(всего_стоек / стоек_из_одной_доски)`.

**Обвязка**: каждая полоса обвязки = `ceil(длина_стены / 6000)` досок. Всего полос: 1 нижняя + 1-2 верхних.

**Ригель стены**: `ceil(длина_стены / 6000)` досок.

**Проемы**: оптимальный раскрой (см. ниже).

### Алгоритм раскроя досок (bin-packing)

Для элементов проемов (перемычки, ригели проемов, подоконники) применяется алгоритм **First Fit Decreasing** (FFD):

1. **Сбор заготовок**. Для каждого проема формируется список кусков с длиной, равной ширине проема: перемычка (всегда), ригель (если включен), подоконник (если окно).

2. **Сортировка по убыванию**. Все куски сортируются от самого длинного к самому короткому.

3. **Размещение**. Каждый кусок последовательно пробуется в уже открытые 6-метровые доски:
   - если кусок помещается в остаток доски - размещается туда
   - если не помещается ни в одну - открывается новая 6м доска

4. **Результат** - минимальное (или близкое к минимальному) количество 6-метровых досок и карта раскроя с остатками.

Пример: 3 перемычки по 1200 мм + 2 подоконника по 1200 мм = 5 кусков. FFD упакует их в 1 доску (5 × 1200 = 6000, остаток 0).

Визуализация раскроя отображается отдельной SVG-схемой с цветовой кодировкой: перемычки (оранжевый), ригели (коричневый), подоконники (голубой).

### Расчет гвоздей

| Узел | Формула |
|---|---|
| Стойки к нижней обвязке | 2 гвоздя × количество стоек |
| Стойки к верхней обвязке | 2 гвоздя × количество стоек |
| Двойная верхняя обвязка | ceil(длина / 400) + 2 |
| Перемычки | 4 гвоздя × количество перемычек |
| Ригели проемов | 4 гвоздя × количество ригелей |
| Подоконники | 4 гвоздя × количество подоконников |
| Ригель стены | 2 гвоздя × количество стоек |

Итог увеличивается на **15%** (запас на брак). Длина гвоздя = `ceil(толщина_доски × 2 / 10) × 10`.

### Объем пиломатериала

Для каждого элемента: `толщина × ширина × длина` (в метрах). Суммируется по всем стойкам, обвязкам, ригелям, перемычкам и подоконникам.

## SVG-чертеж

Чертеж строится в масштабе с автоматическим подбором под ширину контейнера (max 1100px). Включает:

- Верхнюю и нижнюю обвязки (темнее)
- Стойки с текстурной штриховкой
- Проемы с цветовой кодировкой (двери - коричневый контур, окна - синий)
- Стойки обрамления проемов (темнее основных)
- Перемычки, ригели, подоконники
- Горизонтальные размерные линии между стойками
- Общий размер стены
- Вертикальные размеры (высота стены, высота стоек)
- Кумулятивную шкалу снизу с фильтрацией перекрывающихся меток

## Смета

Формируется при указании стоимости доски 6м. Позиции: стойки, обвязка, ригель стены, проемы. Каждая позиция = количество досок × цена.
